# Actividad de aprendizaje 2 - Programación con Lenguaje Java 

Este proyecto contiene el código fuente con el desarrollo de la actividad

## Requisitos

- Java 11+
- Maven

## Como ejecutar

```bash
mvn exec:java -q -D exec.mainClass="com.App"
```

## Estructura

- `src/main/java/com/App.java` - Clase principal
- `src/main/java/com/Product.java` - Clase de Producto
- `src/test/java/com/AppTest.java` - Pruebas unitarias
